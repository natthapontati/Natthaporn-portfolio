Attendance Check Application

A Python + SQL-based academic tracking system

Goal

To build a student attendance and grade tracking system for instructors and students without a GUI, but with database integration for realistic simulation.

Features

  • Student and instructor login system
  • Instructors can search for students, input scores, and adjust grading criteria
  • Students can check scores and attendance records
  • SQL database used to store and manage all data

Result

The app successfully simulates a real-world education platform backend using pure Python and SQL, offering modular, testable, and scalable code.
